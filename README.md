# Star Trek: Unity

*A Powered by the Apocalypse RPG*

## About

**Star Trek: Unity** is a tabletop roleplaying game set during the final stages of the Dominion War (2375, DS9 Season 6 era). Using the Powered by the Apocalypse engine, it blends the humor and heart of *Lower Decks* with the dramatic stakes of *Voyager*.

## What You Need to Play

- 3-6 players (1 GM + 2-5 players)
- Two six-sided dice (2d6) per player
- Character sheets (printable from the website)
- Ship sheet (printable from the website)
- 3-4 hours per session

## Game Structure

### Stats
- **Protocol** - Starfleet training, command, regulations
- **Science** - Analysis, technical knowledge, understanding
- **Daring** - Bold action, improvisation, combat
- **Heart** - Empathy, morale, connection

### Core Mechanics
- **Cohesion** - Crew unity (shared resource)
- **Pressure** - Strain from war, damage, scarcity (countdown clock)
- **Coordinate** - Collaborative ship action system

## Character Creation

1. Choose a **Role** (your job on the ship)
2. Choose a **Species** (biological/cultural traits)
3. Choose a **Stat Array**
4. Select your **Starting Moves**
5. Answer **character questions** to establish relationships

## Ship Building

Create your ship collaboratively:
- Choose a ship class
- Answer story questions together
- Assign personality traits
- Establish quirks and history
- Record starting systems
